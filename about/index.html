About This Lab

A chaotic experimental space for code, visuals, and micro‑projects.

What is this?

This site is a modular, auto‑generated playground for creative coding. Projects are loaded dynamically from JSON, generated by Python, and rendered with interactive JavaScript effects.

Why?

Because static portfolios are boring. This one is alive — glitching, shuffling, generating, and evolving as new experiments are added.

How it works

  • Python generates project data
  • JavaScript loads and animates everything
  • CSS handles the neon cyber aesthetic
  • GitHub Pages hosts it all
